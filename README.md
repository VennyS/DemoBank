## Описание

DemoBank — это Android‑приложение, позволяющее просматривать банковские карты и их балансы, управлять транзакциями и выполнять переводы.  
Приложение использует локальную базу данных и не требует авторизации, поэтому подходит как учебный и демонстрационный проект по мобильной разработке.

Основные возможности:

- просмотр списка карт с актуальным балансом
- история транзакций с фильтрацией по карте
- переводы между собственными картами
- переводы по номеру телефона с выбором получателя из контактов
- добавление и удаление виртуальных карт

## Технологии

- **Язык:** Kotlin
- **UI:** Jetpack Compose + Material 3
- **Архитектура:** MVVM + Repository Pattern
- **База данных:** Room (2 таблицы: `cards` и `transactions`, связь 1:N с ON DELETE CASCADE)
- **Навигация:** Navigation Compose
- **Асинхронность:** Kotlin Coroutines + Flow/StateFlow

## Требования

- Android 12 (API 31) и выше
- Разрешение `READ_CONTACTS` для перевода по номеру телефона (доступ к контактам)

## Запуск проекта

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Синхронизируйте Gradle
4. Запустите приложение на эмуляторе или физическом устройстве
